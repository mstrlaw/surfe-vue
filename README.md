# surfe-vue

Frontend challenge for Surfe.

### Running the project
1. `cd` into `surfe-vue`
2. Install dependencies with `npm i`
3. Run it with `npm run dev`

### Build it

```sh
npm run build
```

### Run Unit Tests

```sh
npm run test:unit
```

### Lint

```sh
npm run lint
```
